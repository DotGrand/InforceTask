# InforceTask "WebApp"

WebApp is a web application that allows users to login, register, and update their personal information. It provides a simple and user-friendly interface for managing user data.

## Features

- User Registration: Users can create a new account by providing their desired username, email address, and password.
- User Login: Registered users can securely log in to their accounts using their credentials.
- Update User Data: Once logged in, users have the ability to update their personal information such as username, email address, and password.

## Usage

Once you have the WebApp up and running, you can perform the following actions:

- **Registration**: Click on the "Register" link on the homepage and fill in the required fields to create a new account.

- **Login**: Use the "Login" link on the homepage to access the login page. Enter your registered email and password to log in.

- **Updating Data**: After logging in, you will be redirected to your account page. Here, you can update your personal information by editing the relevant fields.

## Technologies Used
- Back-end:
  - Java 8
  - MySQL
  - Spring Web MVC
  - Spring ORM
  - Spring Security
  - Hibernate
  - Javax Servlet
  - JSP
  - JSTL
  - Log4j

- Front-end:
    - HTML
    - CSS
